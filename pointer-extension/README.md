# Pointer Extension

This is a placeholder for a future Chrome MV3 browser adapter.

Week 1 uses the Python desktop `pointer-agent` because OS-level cursor, focused-window,
accessibility-tree, and selected-text capture are the milestone. The extension path is
kept explicit because the Jointer brief also calls out Chrome DOM actions as a later
action-layer option.

No functional extension code is implemented in Week 1.
